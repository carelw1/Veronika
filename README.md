# Инструкции по игре в интерактивную кошачью игру

## Как играть
1. Загрузите игру из репозитория.
2. Убедитесь, что у вас установлены необходимые зависимости.
3. Запустите игру и следуйте инструкциям на экране.

## Как включить GitHub Pages
1. Перейдите в раздел "Настройки" вашего репозитория.
2. Прокрутите вниз до секции "GitHub Pages".
3. Выберите ветку и нажмите "Сохранить".

## Возможности игры
- Игра в интерактивном формате.
- Возможность взаимодействовать с кошками.
- Разнообразные уровни сложности.
- Настройки для изменения внешнего вида.
